Formatações

Negrito / Destaque

Para colocar um termo em NEGRITO basta usar a tag <strong> </strong>

Italico / Deitado

Para colocar um termo em ITÁLICO basta usar a tag <em> </em>

Texto marcado

Para destacar um termo basta usar a tag <mark> </mark>

Texto grande e pequeno

Para aumentar e diminiuir, agora é com CSS3, porem a tag <small> </small> continua para casos especiicos

Texto deletado

Para marcar um texto como excluido usamos a tag <del> </del>

Texto inserido

Para dizer que um texto foi inserido, usamos a tag <ins> </ins>

Texto sobrescrito

Para escrever coisas do tipo f(x) = ax, uitlizamos a tag <sup> </sup>

Texto subescrito

Para escrever coisas do tipo log28 = 3 utilizamos a tag <sub> </sub>

Outras formatações

Codigos

Podemos colocar a tag <code> </code> para deixar mensagens em formas de códigos, mas para arrumar a enedentação, o proprio <code> </code> deve estar dentro de uma tag <pre> </pre>

Citações

no lugas de aspas manuais utilizar a tag <q> </q>

Caso queira colocar uma citação completa, como esta, utilizar <blockquote> </blockquote>

Abreviações

Para dar significado a abreviações, como fml, sdds, utilizar <abbr> </abbr>
